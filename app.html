Software Requirements Specification for Career Guidance Chatbot
Customer Needs Solution Components Budget
Provide career advice based on user interests and skills AI-driven chatbot that analyzes user input to recommend career paths Natural Language Processing (NLP), Machine Learning (ML) algorithms, Database of career options High

Ensure user data privacy and security Implement end-to-end encryption and secure authentication protocols Security protocols, Encryption tools, Authentication service Low
